Welcome to HTML Page

1.What is HTML?

  • HTML stands for HyperText Markup Language. It is the core language used to create the structure of web pages.
  • HTML describes how content such as text, images, links, videos, and forms should be displayed in a browser.
  • It uses elements (tags) enclosed in angle brackets, like html,head,body etc. HTML is not a programming language it's a markup language that structures content.

2.What are HTML tags?

  • HTML tags are the building blocks of an HTML page.
  • They define elements such as headings, paragraphs, links, lists, images, and more.
  • Tags usually come in pairs: an opening tag (e.g.,paragraph tag) and a closing tag. Everything between the tags is the content. Some tags are self-closing, like image or break

3.Difference between div tag and span tag?

  • Div: Div is a block-level container used to group large chunks of HTML elements for layout or styling purposes.
  • Span:Span is an inline-level container used to group small portions of text within a line.
    • Both tags do not add styling by default but are useful for applying styles or scripts.

4.Purpose of alt attribute in images?

  • The alt (alternative text) attribute provides a textual description of an image, which is displayed if the image fails to load.
  • It also assists visually impaired users who rely on screen readers.
  • It’s a best practice for accessibility and SEO.

5.Types of list in HTML?

  1. Ordered List (ol) – displays items with numbers.
  2. Unordered List (ul) – displays items with bullets.
  3. Description List (dl) – used for terms and their descriptions, using dt and dd tags.

6.What are self-closing tags?

  • Self-closing tags are tags that don’t require a closing tag because they do not have any content inside them.
  • These are also known as void elements.
    • Ex:Break tag,Image tag,Horizontal line,input field

7.Difference between id and class

  • Id:he id attribute is used to uniquely identify a single HTML element. No two elements should have the same ID.
  • Class:The class attribute can be used on multiple elements, helping to group and style them together.
